TXT
 1. Создать внешний репозиторий c названием TXT.                                                                                                                                            | GitHub -> Create New Repository TXT
 2. Клонировать репозиторий TXT на локальный компьютер.                                                                                                                                     | git clone https://github.com/sergnn/TXT.git
 3. Внутри локального TXT создать файл “new.txt”.                                                                                                                                           | cd TXT && touch new.txt
 4. Добавить файл под гит.                                                                                                                                                                  | git add touch new.txt
 5. Закоммитить файл.                                                                                                                                                                       | git commit -m "file_txt"
 6. Отправить файл на внешний GitHub репозиторий.                                                                                                                                           | git push
 7. Отредактировать содержание файла “new.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате TXT.            | vim new.txt
                                                                                                                                                                                            | NAME Nabokin Sergey
                                                                                                                                                                                            | AGE 28
                                                                                                                                                                                            | PETS 0
                                                                                                                                                                                            | SALARY 1000 
 8. Отправить изменения на внешний репозиторий.                                                                                                                                             | git commit -am "firstfix" && git push
 9. Создать файл preferences.txt                                                                                                                                                            | touch preferences.txt
 10. В файл preferences.txt” добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате TXT.  | vim preferences.txt
                                                                                                                                                                                            |  FILM Harry Potter
                                                                                                                                                                                            |  SERIES University
                                                                                                                                                                                            |  FOOD Cheeseburger
                                                                                                                                                                                            |  SEASON Summer
                                                                                                                                                                                            |  COUNTRY USA
 11. Создать файл sklls.txt добавить информацию о скиллах которые будут изучены на курсе в формате TXT                                                                                      | vim skills.txt
                                                                                                                                                                                            |  SKILLS The best skills
 12. Сделать коммит в одну строку.                                                                                                                                                          | git add . && git commit -m "changes" 
 13. Отправить сразу 2 файла на внешний репозиторий.                                                                                                                                        | git push                                                                                                                                         |
 14. На веб интерфейсе создать файл bug_report.txt.                                                                                                                                         | GitHub -> Create New file -> TXT/bug_report.txt
 15. Сделать Commit changes (сохранить) изменения на веб интерфейсе.                                                                                                                        | GitHub -> Commit changes
 16. На веб интерфейсе модифицировать файл bug_report.txt, добавить баг репорт в формате TXT.                                                                                               | GitHub -> Edit this file
                                                                                                                                                                                            |  SUMMARY In CTR (Click through ratio) ‘Total’ row calculation is wrong
                                                                                                                                                                                            |  PRODUCT Example product
                                                                                                                                                                                            |  VERSION 1.0
                                                                                                                                                                                            |  PLATFORM PC
                                                                                                                                                                                            |  SYSTEM Windows 2000
                                                                                                                                                                                            |  STATUS NEW
                                                                                                                                                                                            |  SEVERITY Major
                                                                                                                                                                                            |  PRIORITY P1
                                                                                                                                                                                            |  COMPONENT Publisher stats
                                                                                                                                                                                            |  STR 1
                                                                                                                                                                                            |  ER wow
 17. Сделать Commit changes (сохранить) изменения на веб интерфейсе.                                                                                                                        | GitHub -> Commit changes
 18. Синхронизировать внешний и локальный репозиторий TXT                                                                                                                                   | git pull
